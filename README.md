# AI Agent Orchestration using LangChain

## Overview
This project demonstrates an AI agent built using LangChain and TinyLlama that can use tools such as a calculator and a weather API.

## Features
- Natural language command processing
- Calculator tool
- Real-time weather API integration
- LLM-based decision making

## Technologies Used
- Python
- LangChain
- TinyLlama
- OpenWeather API
- HuggingFace Transformers

## Example Commands
weather in chennai
5 * 6

## Installation
pip install -r requirements.txt

## Run the program
python tool_agent.py
